What's actually live: bracket progression. This page polls ESPN's public schedule for the Mexico national team every 6 minutes (free, no key required) — once a stage's real opponent is decided, that stage locks to the confirmed matchup (dropdown disabled) instead of a guess-list, and if Mexico loses, a banner says so.

What's not live: the win-probability numbers themselves. A live odds feed (real sportsbook prices) needs a paid/keyed API and a server-side proxy to avoid exposing that key — skipped here to keep this a zero-setup static page. The percentages are model/market estimates as of July 5, 2026 (see baseline notes on each card); ask me in chat to refresh them after a round and I'll pull current numbers and update the file by hand.

How the sim works: each stage draws a random number against your weight for the selected opponent, cascading — lose any stage, the run stops. Bars show what share of N trials reached each stage; the convergence chart reads the same trials as a running average.
